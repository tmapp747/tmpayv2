![TMPay Logo](../public/logo.png)

## Overview

TMPay is a sophisticated online casino e-wallet platform for 747 Casino, providing secure and streamlined financial transactions with an immersive user experience for top-tier casino management users. This platform enables seamless deposits and withdrawals between users' e-wallets and their casino accounts.

## Key Features

- **Secure Authentication**: Multi-level authentication system with casino account verification
- **Multiple Payment Methods**: Support for GCash QR payments, PHPT/USDT crypto, and manual bank transfers
- **Real-time Casino Integration**: Direct fund transfers to and from 747 Casino accounts
- **Multi-currency Support**: Handle PHP, PHPT, and USDT currencies with exchange capabilities
- **Transaction Tracking**: Comprehensive transaction history and status tracking
- **Admin Dashboard**: Management tools for transaction overview and manual payment approvals
- **Responsive Design**: Mobile-first interface with seamless experience across all devices

## Technology Stack

- **Frontend**: React, TypeScript, TailwindCSS, Shadcn UI, Framer Motion, React Three Fiber
- **Backend**: Express.js, Node.js, TypeScript
- **Database**: PostgreSQL with Drizzle ORM
- **Authentication**: Custom auth system with session management
- **API Integration**: 747 Casino API, DirectPay, Paygram
- **Deployment**: Replit hosting
